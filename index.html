Anti Copy Paster (Python) Fall 2023 Research

This is the documentation site for our research to extend the functionality of PyCharm to include the AntiCopyPaster tool

AntiCopyPaster is an open-source plugin for IntelliJ Idea that automatically detects copied and pasted code and suggests extracting that duplicated code into its own method

Our work is to bring this functionality as a plugin to IntelliJ PyCharm for Python

The goal of this project is to create a Python plugin which detects when the user copies and pastes a section of their own code and refactors the project to minimize repetition. So given a set of parameters, the plugin will determine if the pasted code should be flagged for refactoring, notify and give the user the option to approve refactoring and perform said refactoring. This will be achieved through studying and adapting an existing plugin for PyCharm called PyNose. This plugin can be used inside PyCharm to study the test smells of a certain project. When you open a project in PyCharm, a number of inspections will be available.
One of the tools available is duplicate assert. This will underline duplicated assert statements (two assert statements with the same parameters) with a warning. We intend to use the duplication finder code to look for code that is duplicated throughout the entire file.

Alongside the plugin, the project involves the creation of a website which will function both as a repository for information on our project for the client to review, but also as a source of information for any later groups who wish to further iterate on our plugin.
