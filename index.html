Anti Copy Paster (Python) Fall 2023 Research

This is the documentation site for our research to extend the functionality of PyCharm to include the AntiCopyPaster tool.

AntiCopyPaster is an open-source plugin for IntelliJ IDEA that automatically detects copied and pasted code and suggests extracting that duplicate code into its own method.

Our task is to port the funcitonality of this plugin to IntelliJ PyCharm for Python.

The goal of this project is to create a Python plugin which detects when the user copies and pastes a section of their own code, and suggests refactoring to minimize repetition. Given a set of customizable parameters, the plugin will determine if the pasted code should be flagged for refactoring, notify and give the user the option to approve refactoring in a non-invasive manner, and finally execute said refactoring.

A plugin with similar functionality called PyNose already exists within PyCharm, and we will use it as a basis for our project. PyNose can be used inside PyCharm to study the test smells of a certain project. When you open a project in PyCharm, a number of inspections will be available.

One of the tools available is 'duplicate assert'. This will underline repeated assert statements (two assert statements with the same parameters) and warn the user that this is bad coding practice. We intend to use this duplication finder ultility as a starting point to search for code that is copied and pasted throughout the entire file.

This website will function both as a repository for information on our project's progress for the client to review, and as a source of information for any later groups who wish to further iterate on our plugin.
